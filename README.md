Este proyecto implementa un sistema de gestión para un minimarket basado en una arquitectura REST, utilizando Spring Boot para la lógica de negocio y persistencia de datos mediante Hibernate/JPA. La solución expone endpoints estructurados para el control de inventario, categorías y carritos de compra, los cuales se encuentran completamente documentados de forma automática bajo el estándar OpenAPI. El sistema integra Spring Security para la protección de rutas transaccionales y permite la validación interactiva de sus contratos de datos mediante Swagger UI y Postman.
